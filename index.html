PhreshCatch
End-to-end phishing URL detector using 27 hand-engineered features extracted purely from URL structure — no page visits, no DNS lookups, no HTML parsing. Trained on 498K real-world URLs from 6 million Webroot users.
97.74% phishing catch rate · 0.4ms per URL · XGBoost + Optuna
XGBoost Optuna SHAP Feature Engineering Streamlit HuggingFace
Live Demo GitHub
Key Findings
  • F1 Discovered data leakage in PhiUSIIL dataset — URLSimilarityIndex scored exactly 100 for every legitimate URL, making it a label proxy, not a feature.
  • F2 Model trained on PhiUSIIL achieved 99% accuracy but only 23% recall on ISCX — a complete domain shift collapse. Documented why dataset choice is an architectural decision.
  • F3 PathLength became the #1 SHAP feature — short paths are overwhelmingly phishing. Real sites have real content structure.
