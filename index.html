PhreshCatch — Phishing URL Detector

An end-to-end phishing detection system that classifies URLs in under 0.4ms using 27 hand-engineered features. No page visits, no DNS lookups — just pure URL structure analysis. Built on 498K real-world URLs from Webroot telemetry.

Catch Rate 97.74%

Inference Speed 0.4ms per URL
XGBoost Optuna SHAP Feature Engineering Streamlit
🚀 Live Demo 💻 GitHub
Key Discoveries
01 Found data leakage in PhiUSIIL — URLSimilarityIndex was a label proxy, not a feature. Dropped immediately.
02 Same model: 99% accuracy on its own test set, 23% recall on ISCX-2016. Classic domain shift collapse.
03 PathLength became #1 SHAP feature — real sites have real content. Short paths = almost always phishing.

The "Perfect Accuracy" Trap
Achieved 99% accuracy on PhiUSIIL. Turns out, I had data leakage. Now I always check feature-target correlations first.
⏰
The Time Travel Mistake
Used random split on time-series fraud data. Model saw the future. Always use TimeSeriesSplit for temporal data!
🔥
The Overfitting Odyssey
Spent 3 days tuning XGBoost only to realize CatBoost handled categorical features natively. Sometimes simpler is better.
