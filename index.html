🐛 Bug Journal
Mistakes I've made so you don't have to — documented failures that made me a better engineer
🎯
The Perfect Accuracy Trap
99% accuracy on PhiUSIIL. URLSimilarityIndex scored exactly 100.0 for every legitimate URL — a label proxy, not a feature. Dataset abandoned immediately.
⏰
The Time Travel Mistake
Random split on fraud temporal data. Model saw the future. Now I always use time-based splits for temporal data.
🔥
The Encoding Leak
LOO encoding leaked future fraud stats into training. Val→test gap was 0.11. CatBoost's ordered target statistics dropped it to 0.028.
📊
The Wrong Dataset Choice
V1-V28 anonymized data — no lat/long, no merchant, no timestamps. No feature engineering possible. Switched to Sparkov with raw transaction fields.
