● Live ML · Security
PhreshCatch — Phishing URL Detector
An end-to-end phishing detection system that classifies any URL as phishing or benign in under 0.4ms — using nothing but the URL string itself. No page visits, no DNS lookups, no HTML parsing. Built on 27 hand-engineered features extracted purely from URL structure and trained on PhreshPhish, a production-grade dataset from real browsing telemetry of 6 million Webroot users. The model catches 97.74% of phishing URLs at a threshold of 0.57 — deliberately tuned for security contexts where missing a threat costs far more than a false alarm.

Research Findings
F1 Detected data leakage in PhiUSIIL — URLSimilarityIndex returned exactly 100.0 for every legitimate URL, making it a label proxy not a feature. Dropped immediately.
F2 Same model: 99% accuracy on its own test set, 23% phishing recall on ISCX-2016. A textbook domain shift collapse, documented in full.
F3 PathLength became the #1 SHAP feature — it didn't exist in the original feature set. Real sites have real content. Short paths are almost always phishing.

ML · Fintech
Credit Card Fraud Detection
Detecting financial fraud in 1.2 million transactions where only 0.6% are fraudulent. Standard accuracy is meaningless here — predicting everything as legitimate gives 99.4%. This project diagnoses why XGBoost failed (encoding drift across time splits), switches to CatBoost's native ordered statistics, and improves PR-AUC from 0.67 to 0.93 while collapsing the validation-to-test gap from 0.11 to 0.028. Features include geospatial distance, impossible travel detection, and card-level behavioural baselines.

Healthcare ML
Heart Disease Prediction
A clinical ML pipeline predicting heart disease from 13 routine patient features on the Cleveland dataset. The core insight: default 0.5 thresholds optimise for accuracy, not for medicine. A missed positive is a missed diagnosis — a false negative is catastrophically worse than a false alarm. All three models were threshold-tuned to achieve ≥90% recall first, then differentiated by false positive rate. Engineered 10 clinically-motivated features — 2 of the top 3 SHAP predictors were hand-crafted.

IoT · Safety
Gas Leakage Detection System
An automatic gas detection and valve control system that reacts in under 500ms — no human required. The first version hardcoded a threshold that failed in a different room due to ambient air variation. The fix: adaptive calibration that measures its own environment on startup and sets a personalised baseline. That failure taught a lesson about safety systems that hardcode assumptions — a system that cries wolf is almost as dangerous as one that goes silent.
